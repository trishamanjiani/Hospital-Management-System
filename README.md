# Hospital Patient Management System

## Overview

The Hospital Patient Management System is a console-based application developed in C++. It helps manage patient records and maintains a waiting queue using fundamental data structures.

## Features

* Add new patient records
* Display all patients
* Search patients using Patient ID
* Manage patient waiting queue
* Serve patients in FIFO order

## Technologies Used

* C++
* Linked List
* Queue
* Object-Oriented Programming Concepts
* Data Structures

## Concepts Applied

* Structures
* Pointers
* Functions
* Dynamic Memory Allocation
* Queue Operations
* Linked List Traversal
